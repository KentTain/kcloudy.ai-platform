## MODIFIED Requirements

### 需求:角色权限分配

系统 SHALL 支持为角色分配或取消权限。

#### 场景:为角色分配权限

- **当** 管理员点击角色列表中的「分配权限」按钮
- **那么** 系统弹出独立的权限分配弹窗

#### 场景:批量分配权限

- **当** 管理员在弹窗中选择多个权限并保存
- **那么** 系统一次性更新角色权限关联

#### 场景:分配弹窗展示

- **当** 权限分配弹窗打开时
- **那么** 系统显示权限列表、已选统计、全选按钮、保存/取消按钮
