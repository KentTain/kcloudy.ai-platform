2. 参考项目接口设计分析

2.1 消息队列接口（参考实现）

┌─────────────────────────────────────────────────────────────────────────────┐
│                          Messaging 接口层级                                  │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   base.py                                                                   │
│   ├── Message              # 统一消息模型                          │
│   ├── MessageHeaders       # 消息头                        │
│   ├── Acknowledgment       # 消息确认回调                                   │

│                                                                             │
│   pubsub/interfaces.py                                                      │
│   ├── PubSubProducer       # publish(topic, message) → str|None            │
│   └── PubSubConsumer       # subscribe(topic, handler), unsubscribe(topic) │
│                                                                             │
│   interfaces.py                                                             │

│   ├── get_queue_producer() → QueueProducer                                 │

│   └── register_queue_consumer_handlers([HandlerClass])                     │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
关键设计模式：

模板方法模式：send() / publish() 自动设置 id 和 timestamp，委托 do_send() / do_publish() 实现
工厂模式：factory.py 根据配置的 provider 动态创建实例
策略模式：不同的 MQ 实现（Redis/Kafka/RabbitMQ）可互换
2.2 对象存储接口（参考实现）

class OssProvider(ABC, Generic[GenericConfigData]):
    @abstractmethod
    def type(self) -> OssType: ...

    @abstractmethod
    def get_client(self) -> Any: ...
    
    @abstractmethod
    def exists(self, bucket_name: str, object_name: str) -> bool: ...
    
    @abstractmethod
    def upload(self, bucket_name: str, object_name: str, 
               file: str | bytes) -> tuple[bool, str, str]: ...
    
    @abstractmethod
    def download(self, bucket_name: str, object_name: str, 
                 filepath: str) -> bool: ...
    
    @abstractmethod
    def delete(self, bucket_name: str, object_name: str) -> bool: ...
    
    @abstractmethod
    def get_presigned_url(self, bucket_name: str, object_name: str,
                          expires_minutes: int = 60) -> str: ...
特点：

泛型配置类支持不同 Provider 的配置类型
实现包括：MinIO、阿里云、腾讯云
2.3 分布式锁接口（Java参考）

public interface IDistributedLock<T> {
    void DoDistributedLock(String key, Consumer<T> action);
    void DoDistributedLock(String key, TimeSpan acquireTimeout, Consumer<T> action);
    void DoDistributedLock(String key, TimeSpan acquireTimeout,
                           TimeSpan lockTimeOut, Consumer<T> action);
}
特点：

支持超时控制
使用回调模式封装锁的生命周期
