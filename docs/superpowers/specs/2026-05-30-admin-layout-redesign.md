# AdminLayout 布局重设计

## 概述

重新设计 AdminLayout 组件的布局结构和交互体验，提升整体视觉美感和用户操作效率。

## 设计目标

1. **清晰的信息层级**：左右结构分离，功能区域明确
2. **现代化视觉风格**：渐变色、柔和阴影、统一间距
3. **流畅的交互体验**：搜索框动态展开、面板弹出动画

## 布局结构

```
┌─────────────────────────────────────────────────────────────────────┐
│                              整体布局                                │
├──────────────┬──────────────────────────────────────────────────────┤
│              │  顶部导航栏                                          │
│   侧边栏     │  ├── 搜索框（居中）                                  │
│   ├── 租户   │  ├── 通知 / 待办 / 帮助                             │
│   └── 菜单   │  └── 用户面板                                        │
│              ├──────────────────────────────────────────────────────┤
│              │  内容页导航栏                                        │
│              │  ├── 收缩按钮                                        │
│              │  ├── 面包屑                                          │
│              │  └── 水平导航                                        │
│              ├──────────────────────────────────────────────────────┤
│              │  内容页                                              │
└──────────────┴──────────────────────────────────────────────────────┘
```

### 左侧：侧边栏

| 组件 | 说明 |
|------|------|
| 租户切换器 | 显示租户 logo + 名称，点击可切换租户 |
| 菜单导航 | 分组菜单，支持展开/折叠子菜单 |

### 右侧：上下结构

**上半部分 - 顶部导航栏：**

| 组件 | 说明 |
|------|------|
| 搜索框 | 居中显示，动态交互（详见下方） |
| 功能图标 | 通知、待办、帮助按钮 |
| 用户面板 | 头像 + 姓名 + 通信方式 |

**下半部分 - 内容区：**

| 组件 | 说明 |
|------|------|
| 内容页导航栏 | 收缩按钮 + 面包屑 + 水平导航 |
| 内容页 | 页面主体内容 |

## 组件规格

### 1. 搜索框

**位置：** 顶部导航栏居中

**交互状态：**

| 状态 | 宽度 | 说明 |
|------|------|------|
| 默认 | 280px | 缩小状态，显示搜索图标和占位符 |
| 悬停 | 400px | 鼠标移入后平滑变长 |
| 激活 | 400px | 单击后显示搜索结果面板 |

**搜索结果面板：**

- 位置：搜索框正下方，居中对齐
- 内容：
  - 快速操作：新建文档、新建知识库、系统设置
  - 最近访问：用户最近访问的页面列表
- 宽度：与搜索框一致（400px）

### 2. 功能图标

**图标列表：** 通知、待办、帮助

**红点规则：**

| 图标 | 红点条件 |
|------|----------|
| 通知 | 有未读消息时显示 |
| 待办 | 有待处理事项时显示 |
| 帮助 | 无红点 |

**交互：** 悬停显示 tooltip

### 3. 用户面板

**显示内容：** 头像 + 姓名 + 通信方式（邮箱或手机）

**下拉菜单：**

| 菜单项 | 说明 |
|--------|------|
| 切换主题 | 明/暗主题切换 |
| 账号设置 | 跳转账号设置页面 |
| 退出登录 | 红色警示色，点击退出登录 |

### 4. 通知面板

**触发：** 点击通知图标

**布局：**

- 标题栏：显示"通知" + "全部已读"操作
- 消息列表：时间顺序自下而上（最新消息在底部）
- 每条消息包含：图标 + 标题 + 内容摘要 + 时间
- 底部：查看全部通知链接

**消息样式：**

| 类型 | 图标背景色 |
|------|-----------|
| 系统 | 蓝色 (#e8f3ff) |
| 警告 | 黄色 (#fef3c7) |
| 成功 | 绿色 (#dcfce7) |

## 视觉规格

### 颜色

| 语义 | 色值 | 用途 |
|------|------|------|
| 主色 | #1677FF | 主按钮、选中态、链接 |
| 辅色 | #FF5722 | 用户头像渐变 |
| 背景 | #F8FAFC | 页面背景 |
| 卡片 | #FFFFFF | 卡片、侧边栏、顶栏 |
| 边框 | #E2E8F0 | 分割线、边框 |
| 文字 | #1E293B | 正文 |
| 次要文字 | #64748B | 说明文字 |
| 禁用文字 | #94A3B8 | 占位符 |

### 间距

- 基础单位：4px
- 组件内间距：8px / 12px / 16px
- 区域间距：20px

### 圆角

| 元素 | 圆角 |
|------|------|
| 按钮 | 8px |
| 卡片 | 12px |
| 面板 | 12px |
| 输入框 | 10px |

### 阴影

| 类型 | 阴影值 |
|------|--------|
| 轻微 | 0 1px 3px rgba(0,0,0,0.04) |
| 卡片 | 0 1px 3px rgba(0,0,0,0.06) |
| 弹出 | 0 10px 40px rgba(0,0,0,0.15) |

## 组件改动范围

### 需要修改的文件

| 文件 | 改动说明 |
|------|----------|
| `AdminLayout.vue` | 重构布局结构 |
| `AppNavbar.vue` | 移除收缩按钮，调整为顶部导航 |
| `AppSidebarFooter.vue` | 移除用户面板（移到顶部导航） |
| `AppNavMain.vue` | 保持不变 |

### 需要新增的文件

| 文件 | 说明 |
|------|------|
| `AppSearchBox.vue` | 搜索框组件（动态展开 + 结果面板） |
| `AppNotification.vue` | 通知面板组件 |
| `AppHeaderRight.vue` | 顶部导航右侧区域（功能图标 + 用户面板） |
| `AppContentHeader.vue` | 内容页导航栏（收缩按钮 + 面包屑 + 水平导航） |

### 需要新增的 Store

| Store | 说明 |
|-------|------|
| `useNotificationStore` | 通知状态管理 |

## 验证页面

创建一个无需登录的预览页面 `/preview/layout`，用于验证布局效果。

**路由配置：**

```typescript
{
  path: '/preview/layout',
  name: 'PreviewLayout',
  component: () => import('@/framework/pages/PreviewLayoutPage.vue'),
  meta: { requiresAuth: false }
}
```

**页面内容：** 模拟知识库列表页面，展示完整布局效果。
