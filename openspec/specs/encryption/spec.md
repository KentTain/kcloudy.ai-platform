## 需求
### 需求:系统支持 AES 加密

系统必须提供 AES 加密算法实现，支持 CBC 模式和 PKCS7 填充。

#### 场景:AES 加密成功

- **当** 调用 AES 加密方法，传入明文和密钥
- **那么** 系统返回 Base64 编码的密文

#### 场景:AES 解密成功

- **当** 调用 AES 解密方法，传入密文和密钥
- **那么** 系统返回原始明文

#### 场景:AES 解密失败

- **当** 调用 AES 解密方法，传入无效密文或错误密钥
- **那么** 系统抛出 DecryptionError 异常

### 需求:系统支持 RSA 加密

系统必须提供 RSA 非对称加密算法实现，支持公钥加密、私钥解密。

#### 场景:RSA 公钥加密成功

- **当** 调用 RSA 加密方法，传入明文和公钥
- **那么** 系统返回 Base64 编码的密文

#### 场景:RSA 私钥解密成功

- **当** 调用 RSA 解密方法，传入密文和私钥
- **那么** 系统返回原始明文

### 需求:系统提供加密管理器

系统必须提供加密管理器（EncryptionManager），支持管理多个加密实例，按名称获取加密器。

#### 场景:初始化加密管理器

- **当** 应用启动时，根据 YAML 配置初始化加密管理器
- **那么** 所有配置的加密实例均可通过名称获取

#### 场景:获取不存在的加密实例

- **当** 通过名称获取未配置的加密实例
- **那么** 系统抛出 EncryptionInstanceNotFoundError 异常

### 需求:系统支持落库加密

系统必须支持数据库敏感字段加密，在写入数据库前自动加密，读取后自动解密。

#### 场景:敏感字段加密存储

- **当** 服务层保存包含敏感字段的数据
- **那么** 敏感字段在入库前被加密

### 需求:系统支持前端输入加密

系统必须支持前端传输数据加密，接收加密数据后自动解密。

#### 场景:前端加密数据接收

- **当** 前端发送加密请求体
- **那么** 后端中间件自动解密请求体

### 需求:系统支持前端输出解密

系统必须支持响应数据加密，使用临时密钥加密响应数据，前端可解密。

#### 场景:生成临时密钥

- **当** 前端请求临时密钥
- **那么** 系统返回临时公钥和密钥标识

#### 场景:加密响应数据

- **当** 服务返回敏感响应数据
- **那么** 系统使用临时密钥加密响应
