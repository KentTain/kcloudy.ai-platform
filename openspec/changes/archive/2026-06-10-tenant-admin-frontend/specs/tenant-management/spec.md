# 租户管理功能扩展

## 修改需求

### 需求:租户详情页支持 Tab 切换

原需求：租户详情页展示租户基本信息。

修改后：租户详情页必须支持 Tab 切换，包含基本信息、资源绑定、模块分配三个 Tab。

#### 场景:查看租户详情
- **当** 管理员访问 /admin/tenants/{id}
- **那么** 系统显示租户详情页，默认展示基本信息 Tab

#### 场景:切换 Tab
- **当** 管理员点击不同 Tab
- **那么** 页面切换显示对应的内容

#### 场景:基本信息 Tab
- **当** 管理员查看基本信息 Tab
- **那么** 系统显示租户名称、编码、状态、联系人、过期时间等基本信息

#### 场景:资源绑定 Tab
- **当** 管理员查看资源绑定 Tab
- **那么** 系统显示租户的资源绑定情况，支持修改

#### 场景:模块分配 Tab
- **当** 管理员查看模块分配 Tab
- **那么** 系统显示租户已分配的模块列表，支持分配/取消
