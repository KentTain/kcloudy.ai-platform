## 新增需求

### 需求:权限管理页面布局改造
权限管理页面必须采用左侧权限树 + 右侧详情面板的布局，右侧包含 Tabs（角色列表、菜单列表）。

#### 场景:左侧权限树
- **当** 页面加载完成
- **那么** 左侧显示按资源分组的权限树，支持关键词搜索过滤

#### 场景:选中权限节点
- **当** 用户点击左侧某个权限节点
- **那么** 右侧详情面板显示该权限的名称、编码、资源、操作类型、说明

### 需求:角色列表 Tab
权限管理页面的 Tabs 必须包含"角色列表"Tab，显示所有角色及其对应权限。

#### 场景:查看角色列表
- **当** 切换到"角色列表"Tab
- **那么** 以表格显示角色列表：名称、编码、描述、成员数、操作列（分配权限、查看成员）

#### 场景:分配权限
- **当** 点击角色的"分配权限"按钮
- **那么** 打开权限分配对话框，以 Checkbox 树形式展示权限，已分配的权限默认勾选

### 需求:菜单列表 Tab
权限管理页面的 Tabs 必须包含"菜单列表"Tab，显示菜单树结构。

#### 场景:查看菜单列表
- **当** 切换到"菜单列表"Tab
- **那么** 以树形结构显示菜单列表，支持展开折叠
