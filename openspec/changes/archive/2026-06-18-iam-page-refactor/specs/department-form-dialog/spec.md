## 新增需求

### 需求:部门创建编辑弹窗
系统必须提供 DepartmentFormDialog 组件，用于创建和编辑部门。

#### 场景:创建一级部门
- **当** 组织树为空时，点击"新增一级组织"按钮
- **那么** 弹窗打开，上级部门字段为空，保存后在组织树中创建根级节点

#### 场景:创建子部门
- **当** 选中一个组织节点后，点击"新增子组织"按钮
- **那么** 弹窗打开，上级部门字段自动填充为当前选中的组织

#### 场景:创建同级部门
- **当** 选中一个组织节点后，点击"新增同级"按钮
- **那么** 弹窗打开，上级部门字段自动填充为当前节点的父组织

#### 场景:编辑部门
- **当** 选中一个组织节点后，点击"编辑"按钮
- **那么** 弹窗打开，表单填充当前组织的名称、编码、排序号、负责人等信息

#### 场景:表单验证
- **当** 提交表单时，名称为空
- **那么** 表单验证失败，显示"请输入部门名称"错误提示

#### 场景:选择上级部门
- **当** 弹窗中的上级部门字段使用 TreeSelect 组件
- **那么** 下拉显示组织树，不允许选择自身或其子孙节点作为父组织
