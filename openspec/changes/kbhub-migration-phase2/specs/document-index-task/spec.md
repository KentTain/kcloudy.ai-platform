# 文档切片索引任务

## 场景

### 触发文档切片任务

- **Given** 文件上传完成且格式支持切片（PDF/Word/TXT/Markdown等）
- **When** 系统自动触发或用户手动触发切片任务
- **Then** 创建切片任务记录，状态为 pending，加入任务队列

### 触发文档切片任务——不支持格式

- **Given** 文件格式不支持切片（如图片、视频）
- **When** 文件上传完成
- **Then** 不触发切片任务，文档索引状态为 skipped

### 切片参数配置——文档库级

- **Given** 文档库配置了切片参数（切片最大长度、重合长度）
- **When** 执行切片任务
- **Then** 使用文档库级参数进行切片

### 切片参数配置——知识库覆盖

- **Given** 知识库配置了独立的切片参数
- **When** 为知识库文档执行切片
- **Then** 使用知识库级参数覆盖文档库参数

### 切片参数校验

- **Given** 配置切片参数
- **When** 最大重合长度 >= 切片最大长度
- **Then** 系统拒绝保存并返回"重合长度必须小于切片长度"的错误提示

### 切片任务执行——成功

- **Given** 切片任务处于 pending 状态
- **When** 任务执行成功
- **Then** 文档切片结果保存，生成向量索引，任务状态为 completed

### 切片任务执行——失败

- **Given** 切片任务处于 pending 状态
- **When** 任务执行失败（如文件损坏、解析失败）
- **Then** 任务状态为 failed，记录错误信息，可重试

### 切片任务重试

- **Given** 切片任务状态为 failed
- **When** 用户手动重试
- **Then** 重新创建 pending 任务，重新执行切片

### 切片任务进度查询

- **Given** 切片任务正在执行
- **When** 用户查询任务状态
- **Then** 返回任务状态、进度百分比、已处理片段数

### 批量触发切片任务

- **Given** 文档库中存在多个未索引文档
- **When** 用户批量触发切片任务
- **Then** 为每个文档创建独立的切片任务，并行执行

### 文档更新触发重切片

- **Given** 文档已索引
- **When** 文档内容更新（新版本上传）
- **Then** 自动触发重切片任务，旧索引标记为失效

### 删除文档清理索引

- **Given** 文档已索引
- **When** 文档软删除或永久删除
- **Then** 关联的切片和索引标记为失效或删除

### 切片任务优先级

- **Given** 任务队列中存在多个切片任务
- **When** 按优先级调度
- **Then** 手动触发的任务优先于自动触发的任务

### 索引恢复补偿

- **Given** 切片任务完成但索引失败
- **When** 定时任务检测到索引状态不一致
- **Then** 自动触发索引恢复任务

### 向量维度配置

- **Given** 系统配置了向量维度（默认 1024）
- **When** 执行切片任务生成向量
- **Then** 向量维度与系统配置一致

### 多租户隔离

- **Given** 不同租户的切片任务
- **When** 查询任务状态
- **Then** 仅返回当前租户的任务记录
