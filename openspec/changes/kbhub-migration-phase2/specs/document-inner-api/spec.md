# document 内部接口规范

## 场景

### 回查用户对源文件资源权限——editable

- **Given** 用户对源文件拥有 editable 权限
- **When** 内部模块调用权限回查接口，传入用户 ID 和源文件 ID
- **Then** 接口返回 editable

### 回查用户对源文件资源权限——readonly

- **Given** 用户对源文件拥有 readonly 权限
- **When** 内部模块调用权限回查接口，传入用户 ID 和源文件 ID
- **Then** 接口返回 readonly

### 回查用户对源文件资源权限——none

- **Given** 用户对源文件无任何权限
- **When** 内部模块调用权限回查接口，传入用户 ID 和源文件 ID
- **Then** 接口返回 none

### 回查用户对源文件资源权限——deny

- **Given** 用户对源文件被显式 deny
- **When** 内部模块调用权限回查接口，传入用户 ID 和源文件 ID
- **Then** 接口返回 none（deny 在权限等级中等同于无权限）

### 回查文档库成员身份——是成员

- **Given** 用户是文档库的成员
- **When** 内部模块调用成员身份回查接口，传入用户 ID 和文档库 ID
- **Then** 接口返回 true 及该成员的角色信息

### 回查文档库成员身份——非成员

- **Given** 用户不是文档库的成员
- **When** 内部模块调用成员身份回查接口，传入用户 ID 和文档库 ID
- **Then** 接口返回 false

### 无认证——仅供模块间调用

- **Given** 内部接口不要求外部认证（无 Bearer Token）
- **When** 内部模块直接调用该接口
- **Then** 接口正常响应，返回查询结果

### 无认证——外部请求拒绝

- **Given** 内部接口设计为仅供模块间调用
- **When** 外部请求（非内部模块）尝试调用该接口
- **Then** 系统拒绝请求并返回 403 错误

### 源文件不存在返回 404

- **Given** 传入的源文件 ID 在系统中不存在
- **When** 内部模块调用权限回查接口
- **Then** 接口返回 404 状态码及"源文件不存在"的错误信息

### 文档库不存在返回 404

- **Given** 传入的文档库 ID 在系统中不存在
- **When** 内部模块调用成员身份回查接口
- **Then** 接口返回 404 状态码及"文档库不存在"的错误信息

### 跨租户数据不可访问——权限回查

- **Given** 用户属于租户 A，源文件属于租户 B
- **When** 内部模块调用权限回查接口，传入用户 ID 和源文件 ID
- **Then** 接口返回 none，不泄露跨租户资源的存在性

### 跨租户数据不可访问——成员身份回查

- **Given** 用户属于租户 A，文档库属于租户 B
- **When** 内部模块调用成员身份回查接口，传入用户 ID 和文档库 ID
- **Then** 接口返回 false，不泄露跨租户资源的存在性
